## CyberteksIT — Company Website

Modern, professional marketing site for CyberteksIT built with **React**, **Vite**, **Tailwind CSS**, **Framer Motion**, **GSAP**, and **Three.js** (via `@react-three/fiber` and `@react-three/drei`).

### Running the project

```bash
npm install
npm run dev
```

Then open the printed local URL (typically `http://localhost:5173`).

### Structure

- `src/App.jsx` — routes and page layout
- `src/components` — shared UI components such as `Navbar`, `Layout`, `Hero3DBackground`
- `src/pages` — individual pages: Home, About, Services, Products, Careers, Contact, Get Started
